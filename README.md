# Ersisk Tap

## How do I install these formulae?

`brew install ersisk/tap/<formula>`

Or `brew tap ersisk/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "ersisk/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
